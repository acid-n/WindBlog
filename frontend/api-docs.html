WindBlog - Backend API

Примечание: Фронтенд-приложение находится в разработке. Вы можете взаимодействовать с бэкенд API напрямую через следующие эндпоинты.

Доступные API эндпоинты Доступно

Метод Эндпоинт Описание
GET /api/v1/posts/ Получить список всех опубликованных постов
GET /api/v1/posts/{id}/ Получить детальную информацию о посте по ID
GET /api/v1/tags/ Получить список всех тегов
GET /api/v1/tags/{slug}/posts/ Получить все посты с определенным тегом
GET /api/v1/archive/years/ Получить архив по годам
GET /api/v1/archive/{year}/months/ Получить архив по месяцам для указанного года
POST /api/v1/contact/ Отправить контактное сообщение
GET /api/v1/ratings/posts/{post_id}/ Получить рейтинги для поста

Примеры использования API Примеры

Получение списка постов:

curl -X GET http://localhost:8000/api/v1/posts/ -H "Content-Type: application/json"

Получение поста по ID:

curl -X GET http://localhost:8000/api/v1/posts/1/ -H "Content-Type: application/json"

Поиск постов:

curl -X GET "http://localhost:8000/api/v1/posts/?search=python" -H "Content-Type: application/json"

Отправка контактного сообщения:

curl -X POST http://localhost:8000/api/v1/contact/ \
-H "Content-Type: application/json" \
-d '{"name": "Иван Иванов", "email": "ivan@example.com", "message": "Привет! Отличный блог!"}'

Административный интерфейс:

Для доступа к административному интерфейсу Django перейдите по адресу: http://localhost:8000/admin/
